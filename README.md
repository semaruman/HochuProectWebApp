## 🛠 Хочу проект — Фриланс-биржа для инженеров

## 🚀 О проекте
Хочу проект — это бэкенд-ядро (на данный момент) для специализированной биржи фриланса, ориентированной на инженерные задачи: проектирование, расчёты, CAD-моделирование, написание технической документации и т.д.

Заказчики создают детализированные объявления, исполнители откликаются, договариваются о сделке и безопасно завершают проекты.

>❗ Backend приложения готов, однако фронтенд отсутствует.

## ✨ Возможности API

### Для клиентов:
- 🔍 **Просмотр всех объявлений** - список всех объявлений в системе
- 📋 **Категории объявлений** - просмотр объявлений по конкретной категории
- 🛒 **Создание объявлений**
- 📊 **Получение всех категорий**
- 
### Для администраторов:
- ➕ **Добавление категории**
- ✏️ **Удаление категории**

## Технологический стек

| Компонент | Технология |
|-----------|-------------|
| **C#** | .NET 8  |
| **ORM** | Entity Framework Core (Linq, Migrations) |
| **Архитектура** | ASP.NET Core WEB API controller-based |
| **Логирование** | Кастомный LoggingMiddleware + Serilog  |
| **Обработка исключений** | Кастомный ExceptionHandler для отлова всех исключений |
| **API стиль** | REST |
| **База данных** | Postgres Sql |
| **Авторизация** | Claims, Cookies |
| **Swagger** | Отправка запросов на сервер |
